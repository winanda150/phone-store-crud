# 📱 Phone Store CRUD - Create, Read, Update, dan Delete

![Phone Store CRUD Dashboard](docs/assets/img/Screenshot%20(198).png)

Aplikasi CRUD Toko HP berbasis web yang dirancang untuk mengelola data, transaksi penjualan dan pembelian, laporan, serta dashboard interaktif. Phone Store CRUD merupakan aplikasi berbasis web yang dikembangkan sebagai Project Ujian Akhir Semester (UAS) pada mata kuliah Pemrograman Web.

Project ini mengangkat studi kasus sebuah Toko HP dengan tujuan mengimplementasikan konsep Create, Read, Update, dan Delete (CRUD) dalam sebuah sistem informasi berbasis web. Aplikasi ini dirancang untuk membantu proses pengelolaan data toko secara lebih efektif, terstruktur, dan mudah digunakan dibandingkan dengan proses administrasi secara manual.

Selain mengimplementasikan operasi CRUD sebagai fitur utama, aplikasi ini juga menyediakan berbagai fitur pendukung seperti dashboard interaktif, pengelolaan data produk, manajemen pengguna, transaksi pembelian, transaksi penjualan, laporan, serta visualisasi data dalam bentuk grafik yang membantu pengguna memantau aktivitas toko secara lebih informatif.

---

## ✨ Fitur Utama

- 👥 Manajemen data pengguna
- 🛒 Pengelolaan transaksi pembelian
- 💰 Pengelolaan transaksi penjualan
- 📄 Laporan transaksi penjualan
- 📊 Dashboard interaktif
- 📈 Visualisasi data menggunakan grafik
- 🔍 Pencarian data
- ✏️ Operasi Create, Read, Update, dan Delete (CRUD)
- 📱 Tampilan dashboard yang responsif

## 🎯 Tujuan Pengembangan

Project ini dikembangkan untuk:

- Memenuhi Project Ujian Akhir Semester (UAS) mata kuliah Pemrograman Web.
- Mengimplementasikan konsep Create, Read, Update, dan Delete (CRUD) pada aplikasi berbasis web.
- Membangun sistem pengelolaan data Toko HP menggunakan PHP dan MySQL.
- Mengembangkan dashboard interaktif sebagai media penyajian informasi.
- Menerapkan konsep pengelolaan transaksi penjualan dan pembelian.
